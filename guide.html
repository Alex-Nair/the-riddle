The Riddle - Guidebook

If you're having trouble with a part of the riddle, then this page is for you.

The guidebook, as the name suggests, has all of the solutions to each of the puzzles. As always, I highly suggest that you attempt to solve each puzzle by yourself before you rely on the guidebook. However, if you're truly stumped and have no idea of what to do next, then this guidebook should hopefully save you time searching for a solution online that may or may not exist.

Each answer will be written in a progressive way. Any important codes will usually be put at the end of paragraphs in blue, and anything before that will be hints and the process. That way, if you just need a helping hand, you won't have the answer spoilt for you if you read from top-to-bottom.

Another thing. Since this program uses other forms of media (like videos or games), I'll also include images in this guidebook so that it's more clear what you need to do. I recommend reading this guidebook after full-sccreening your browser if you're on a desktop, since that will let you see the images as large as possible.

The Riddle - Opening

The opening part of The Riddle has you solve a random question just to get you into the flow of things. Though I must admit, the answer to this one can be a bit obscure. Remember how you got this one, it'll show up a few more times throughout The Riddle past this point.

The first step is obvious. Click the button. There's not much else you can do here. In fact, in order to get to this guidebook, you had to click the button in order to get the link.

The next step requires some searching. In fact, if you input the wrong code, you'll be outright told this step. A common trick is colouring text to be the same colour as the background, making it invisible. However, if you highlight that text, then you can see it. An easy way to find text like this is to hit "Ctrl + A". Doing this will reveal some text in the bottom-right corner of the page.

The text here is in the form of a code. Part #1, then an underscore, then part #2. So if #1 was "BEEF", and #2 was "STEW", the final code is "BEEF_STEW". Of course, those aren't the answers.

The first part is literally given to us, "FROM". So the code currently is "FROM_???". That second part has some instructions that we need to follow.

Taking the first letter of the first four paragraphs and then reversing it gives "ANTI". Therefore, the full code is "FROM_ANTI".

With this code, click "Start", put it into the prompt, and you're done.
